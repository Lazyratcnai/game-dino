# 🦕 恐龙远徙之谜

一个面向10-12岁儿童的浏览器探索解谜+塔防建设游戏。

## 🎮 游戏介绍

小公主被父王派往封地接受继承测试。在建设城镇的过程中，她发现了远古恐龙文明留下的秘密——恐龙没有灭绝，它们在6500万年前集体迁徙去了远方。玩家将通过探索遗迹、解开谜题、建设城镇、塔防战斗，一步步揭开真相。

## 🛠 技术栈

- 纯浏览器实现（HTML5 + Canvas + Vanilla JS）
- 零依赖，零安装
- 响应式设计，支持PC和移动端

## 🚀 运行方法

直接在浏览器中打开 `index.html` 即可。

推荐 Chrome / Firefox / Edge 最新版。

## 📁 文件结构

```
game-dino/
├── index.html   # 主页面 + CSS + 界面结构
├── data.js      # 游戏数据（章节、NPC、谜题、敌人）
├── game.js      # 游戏引擎（主循环、战斗、建造、对话）
├── SPEC.md      # 设计规格文档
└── README.md    # 本文件
```

## 🎯 核心玩法

| 模式 | 说明 |
|------|------|
| 🔍 探索 | WASD移动，点击与NPC/遗迹互动 |
| 🧩 解谜 | 点击石碑等触发谜题，推动剧情 |
| 🏗️ 建造 | 点击建造按钮，选择建筑放置 |
| ⚔️ 战斗 | 点击战斗模式，击败来袭的怪物 |

## 📖 章节内容

- **序章：踏上封地** - 熟悉基本操作，解开第一个谜题
- **第一章：地底的秘密** - 深入遗迹，挑战塔防战斗
- **第二章：守护者的秘密** - 揭开恐龙文明的真相，选择结局
