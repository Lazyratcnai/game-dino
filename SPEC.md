**版本**: v0.1 原型体验版  
**更新**: 2026-03-22

## 1. 游戏概述

**类型**: 浏览器 RPG / 探索解谜 / 塔防建设  
**目标用户**: 10-12岁儿童  
**平台**: 浏览器优先（HTML + Canvas + Vanilla JS）  
**语言**: 中文

**一句话描述**: 小公主继承远古封地，通过探索遗迹、解开恐龙文明谜题、建设城镇、塔防战斗，一步步揭开6500万年前恐龙集体迁徙的真相。

---

## 2. 故事线

### 核心设定
- 6500万年前，恐龙文明预见到小行星撞击灾难
- 它们没有灭绝，而是集体迁徙去了Kepler-442b星球（112光年外）
- 迁徙计划总设计师"星语者"留守地球，记录一切，等待继承者
- 孩子扮演的小公主，被父王派往封地接受继承测试，意外成为这个继承者

### 三章结构

| 章节 | 名称 | 主线任务 |
|------|------|---------|
| 序章 | 踏上封地 | 探索3处 + 解开1个谜题 + 建造1个建筑 |
| 第一章 | 地底的秘密 | 探索5处 + 解开1个谜题 + 建造2个建筑 + 击败1个敌人 + 收集1物品 |
| 第二章 | 守护者的秘密 | 探索6处 + 解开星语者试炼 + 建造3个建筑 + 击败2个敌人 + 收集1物品 + 双结局选择 |

### 三大结局
1. **新领主结局**: 成为真正领主，继承恐龙文明核心
2. **记录者结局**: 还原历史，成为文明记录者
3. **双结局选择**: 
   - 3A: 向Kepler-442b发射信号，重建两个文明的联系
   - 3B: 继承使命，做新一代守护者

## 3. 核心玩法

### 3.1 探索
- WASD / 方向键移动公主
- 接近物件（60像素内）高亮显示
- 点击物件触发互动：对话、解谜、收集

### 3.2 解谜
- 符号排序谜题（序章石碑）
- 远古机关组装（第一章草图）
- 星语者记忆试炼（第二章祭坛）
- 谜题触发剧情，驱动故事发展

### 3.3 建设
- 资源：💰金币 / 🪵木材 / 🪨石材 / 💎水晶碎片
- 可建造：
  - 🏠 小屋（30💰+20🪵）- 增加人口上限
  - 🌾 农田（20💰+10🪵）- 每回合产出金币
  - 🗼 瞭望塔（50💰+20🪨）- 侦察周围
  - ⚒️ 工坊（40💰+30🪵+10🪨）- 解读符号
  - 🏰 兵营（60💰+30🪨+20🪵）- 训练战士
  - 📡 信号塔（100💰+50🪨+1💎）- 发射星际信号（第二章）

### 3.4 战斗（塔防+英雄）
- 建造城墙/防御塔自动防守
- 小公主可点击敌人释放技能：
  - 🔥 火球术（消耗10MP，造成25伤害）
  - 🛡️ 守护之盾（消耗15MP）
  - ✨ 治愈之光（消耗20MP，恢复30HP）
- 敌人类型：
  - 🟢 史莱姆（HP30，伤害5）
  - 🐺 暗影狼（HP60，伤害12）
  - 🗿 石魔像（HP120，伤害20）

## 4. 居民/NPC系统

- 居民有职业（农民、工匠、学者、战士）
- 部分居民是"守护者"，掌握恐龙知识碎片
- 对话解锁线索，推动剧情
- 好感度系统（初步设计，未完全实现）

---

## 5. 视觉风格

- **卡通风格**（非像素，非3D）
- Canvas 2D 渲染
- 角色：公主 + NPC + 敌人 + 建筑物
- 地图：程序化生成地形格子

---

## 6. 技术架构

### 文件结构
```
game-dino/
├── index.html   # 主页面 + 所有CSS
├── data.js      # 游戏数据（章节、NPC、谜题、敌人、道具）
├── game.js      # 游戏引擎（循环、输入、战斗、对话、建造）
└── SPEC.md      # 本规格文档
```

### 状态机
- `start` → `story` → `playing` ↔ `dialog` ↔ `puzzle`
- `playing` ↔ `battle`
- `battle` → `result` → 下一章 / 重新开始

## 9. 设计决策记录

| 日期 | 决策 | 原因 |
|------|------|------|
| 2026-03-22 | 卡通风格 | 比像素更符合10-12岁审美 |
| 2026-03-22 | 塔防+英雄战斗 | 比纯回合制有趣，比即时动作易实现 |
| 2026-03-22 | 章节制结构 | 有目标感，又不完全线性 |
| 2026-03-22 | 小公主身份 | 有别于常规勇者套路，故事感更强 |
| 2026-03-22 | 神秘居民+解谜结合 | 推动剧情，提供线索 |
| 2026-03-22 | 三结局设计 | 给予孩子选择权，结局有重量感 |
